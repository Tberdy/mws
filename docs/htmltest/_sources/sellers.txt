############
Sellers
############

.. function:: list_marketplace_participations(self)

    Returns a list of marketplaces a seller can participate in and a list of participations that include seller-specific information in that marketplace.
    The operation returns only those marketplaces where the seller's account is in an active state.

    :returns: something from amazon


.. function:: list_marketplace_participations_by_next_token(self, token):

    Takes a "NextToken" and returns the same information as "list_marketplace_participations".
    Based on the "NextToken".

    :param string token:
    :returns: something from amazon
